Написати сайт курсу валют, в якого буде можливість додавати нові курси співвідношення валюти до гривні. Для кожної валюти має бути два значення курсу купівлі та курсу продажу. В результаті отримуємо таблицю із валютами, яка буде мати 3 колонки: назву, курс купівлі та курс продажу.

Додайте можливість збереження результатів поточного дня в окремій вкладці.

Додайте можливість додавання нових валют за допомогою вбудованою адмін-панелі.

Забезпечте доступ по API задля відображення актуального на сьогодні курсу валют.
